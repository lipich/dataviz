Файлы с данными

    Пример файла с данными

    Название данных
    Год;Показатель1;Показатель2
    2000;120.5;12
    2001;144;333
    2002;23;233.4
    2003;122;234
    2004;12.9;53
    2005;1300;123

    - расширение файла - csv
    - максимальный размер файла - 32 мб
    - разделитель данных - ;
    - первая строка содержит название данных
    - вторая строка содержит названия рядов
    - третья строка и далее содержит данные рядов
    - значения первого ряда - годы или целые числовые значения
    - значения остальных рядов - с разделителем дробной части , или .
